# AI Resume Screener - Frontend

This is the React frontend for the AI Resume Screener application.

## Getting Started

### Prerequisites

*   Node.js and npm

### Installation

1.  **Navigate to the frontend directory:**
    ```bash
    cd frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running the Application

1.  **Start the frontend development server:**
    ```bash
    npm start
    ```

The application will be available at `http://localhost:3000`.

## Features

*   Paste a resume into a text area.
*   Submit the resume for analysis to the backend.
*   View the analysis results, including:
    *   Decision (interview or reject).
    *   Recruiter summary.
    *   Interview questions (if applicable).
    *   Rejection reason and improvement suggestions (if applicable).
    *   Extracted skills, experience, strengths, and missing requirements.
